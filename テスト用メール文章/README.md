# テスト用メール文章

AI回答支援アプリのE2E確認用に、回答しやすい問い合わせ文を製品別にまとめたものです。

使い方:

1. AI回答支援アプリで対象製品を選択します。
2. 必要に応じてマニュアル/公式ドキュメント/過去案件を検索します。
3. 検索結果から根拠を選択します。
4. 各ファイル内のメール本文を問い合わせ入力欄へ貼り付けて回答生成します。

注意:

- 本文中の会社名・担当者名・メールアドレスはすべて架空です。
- 現状の検索/RAGでヒットしやすいよう、本文に製品名、エラー語、マニュアルタイトルに近い語句を意図的に含めています。
- `Klcwork` は製品名としては通常 `Klocwork` 表記のため、ファイル名と本文は `Klocwork` で作成しています。
- 最新バージョン系のCheckmarx質問は、CuratedFactCatalog優先の動作確認に使えます。
